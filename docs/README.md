# 로또 미션 기능 명세서

## 프로젝트 설명
- 해당 프로젝트는 사용자가 로또를 구입할 비용을 입력하면, 해당 비용 만큼의 로또 번호를 뽑아주고 당첨 번호와 비교를 할 수도록 해주는 프로젝트입니다.
- 해당 프로젝트의 주요 기능은 다음과 같습니다.
  - 사용자가 구매하고싶은 금액을 입력하면, 금액 / 1000원 만큼의 로또 번호를 뽑아준다.
  - 당첨번호와 보너스 번호를 입력하면, 비교를 통해 당점 통계를 알려준다.


## 주요 기능
### 1. 로또 구매 기능
### 2. 숫자 비교( 등수 판별 ) 기능
### 3. 통계 계산 기능
### 4. 사용자 입력 기능
### 5. 출력 기능
